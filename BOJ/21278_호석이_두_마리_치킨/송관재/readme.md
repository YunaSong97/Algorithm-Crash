# 21278번: 호석이 두 마리 치킨

건물 번호가 낮은 순서대로 2개씩 조합으로 뽑아 bfs를 이용해 각 도시와의 거리를 구해서 최솟값이 되는 두 도시를 뽑으면 되는 문제, 답이 같다면 건물 번호가 낮은 것부터 뽑아야 하므로 조합이 낮은 번호 순으로 돌기 때문에 최솟값이 갱신되는 경우가 아니라면 답으로 뽑지 않는다. bfs를 돌 때 진입점을 조합에서 뽑은 두 점을 동시에 넣고, 거리가 현재 bfs 내에서 저장된 거리 배열보다 적은 값이 나올때만 값을 할당하면 두 도시중 짧은 거리가 자동으로 저장된다.
