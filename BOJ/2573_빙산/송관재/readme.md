# 2573번: 빙산

한 덩어리의 빙산이 몇년 째에 녹아서 두 덩어리 이상이 되는지 구하는 문제, 완전히 녹을때까지 분리되지 않는 경우도 있음, dfs로 덩어리가 얼마나 있는지 구했고, melt라는 메소드를 따로 만들어서 녹았을 때를 구현했음. 처음에 구현을 잘못해서 배열을 처음부터 돌면서 녹았는지를 체크하다 보니 얼음이 녹은 칸에서 영향을 주는 것을 생각을 안했었음. 이런 경우에 녹았다면 일단 음수로 둬서 영향을 주지 않게 했고 마지막에 음수인 모든 값을 0으로 되돌리는 작업을 함.
