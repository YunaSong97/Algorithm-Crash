### # move()
1. 속력과 방향에 따라 배열을 갱신한다.
2. 기존 위치에 저장된 배열값을 삭제한다.
</br></br>

### # cal()
1. 두 개 이상의 파이어볼이 있는 칸이라면 조건에 맞게 구현한 후 배열 값 갱신하고 큐에 넣는다.
2. 하나의 파이어볼만 있는 칸이라면 기존 내용 그대로 큐에 넣는다.
