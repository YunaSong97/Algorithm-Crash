### # main()
1. 팀의 인원이 최소 한명 부터 N명까지 나누는 경우의 수를 만들기 위해 for문을 돌린다.
</br></br>

### # comb()
1. 만약 인원 수 만큼 조합을 다 구했다면 능력치 합의 최소를 구한다.
2. 조합을 이용하여 팀을 구성한다.
</br></br>

### # getSub()
1. 둘 다 방문한 팀이라면 vSum에 능력치를 더한다.
2. 둘 다 방문하지 않은 팀이라면 nVSum에 능력치를 더한다.
3. vSum과 nVSum 차이의 절대값을 구해 리턴한다.
